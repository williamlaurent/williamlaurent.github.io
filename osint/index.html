OSINT Training Roadmap

Master Open Source Intelligence from Basics to Advanced

1. Pengantar OSINT dan Etika

Durasi: 1 Minggu

  • Apa itu OSINT?
  • Pentingnya OSINT di berbagai bidang.
  • Hukum, etika, dan batasan dalam OSINT.
  • Memahami ancaman privasi dan keamanan.
Challenge:

Buat presentasi sederhana tentang peran OSINT dalam menyelesaikan kasus nyata, seperti penelusuran organisasi atau individu untuk tujuan analisis publik.

2. Tools dan Framework Dasar OSINT

Durasi: 1 Minggu

  • Perkenalan tools dasar: Google Dorking, Maltego, dan OSINT Framework.
  • Dasar-dasar Google Dorking: operator pencarian, filter pencarian, dll.
  • Memahami OSINT Framework dan struktur pencarian.
Challenge:

Gunakan Google Dorking untuk menemukan informasi domain publik dari sebuah website pemerintah atau non-profit. Buat laporan hasil pencarian Anda.

3. Pencarian Informasi dari Media Sosial

Durasi: 1 Minggu

  • Menggunakan media sosial untuk OSINT: Twitter, Facebook, Instagram, LinkedIn.
  • Tools seperti Social-Searcher, Twint, dan Followerwonk.
  • Identifikasi pola, hubungan, dan aktivitas melalui media sosial.
Challenge:

Lakukan investigasi sederhana pada profil publik di Twitter atau Instagram. Analisis aktivitas dan pola unggahan mereka.

4. Pemanfaatan Data Publik dan Database Online

Durasi: 1 Minggu

  • Cara menggunakan WHOIS, Shodan, dan Censys.
  • Pemanfaatan dokumen publik (PDF, Excel, dll).
  • Pemahaman metadata dari file digital.
Challenge:

Temukan data domain dan informasi perangkat server menggunakan WHOIS dan Shodan. Presentasikan temuan Anda dalam bentuk infografis.

5. Pemetaan dan Geolokasi

Durasi: 1 Minggu

  • Pengenalan geolokasi dengan Google Earth dan OpenStreetMap.
  • Analisis metadata gambar untuk lokasi (Exiftool).
  • Menggunakan tools seperti Geotagging Viewer.
Challenge:

Berikan gambar yang memiliki metadata lokasi aktif. Minta peserta menentukan lokasi menggunakan alat yang telah diajarkan.

6. Analisis Jejak Digital dan Profiling

Durasi: 1 Minggu

  • Membaca jejak digital seseorang.
  • Memahami teknik rekonstruksi profil individu atau organisasi.
  • Tools tambahan seperti SpiderFoot atau Recon-ng.
Challenge:

Buat profil individu fiktif dari data digital yang tersedia di internet. Laporkan dengan struktur profesional.

7. OSINT Lanjutan: Automasi dan Scripting

Durasi: 1 Minggu

  • Automasi dengan Python: Scrapy, Beautiful Soup.
  • Memahami API dari platform OSINT.
  • Membuat script sederhana untuk pencarian data.
Challenge:

Tulis script Python yang mengambil metadata gambar dari folder dan menyusunnya dalam laporan JSON.

8. Studi Kasus dan Proyek Akhir

Durasi: 1 Minggu

  • Studi kasus nyata (diberikan oleh mentor).
  • Strategi penyelidikan dan pelaporan OSINT secara menyeluruh.
  • Pembuatan laporan OSINT profesional.
Challenge:

Pilih salah satu studi kasus yang diberikan, lakukan penyelidikan lengkap, dan susun laporan akhir dengan semua temuan.
