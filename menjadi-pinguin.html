Basic Linux Commands

Command Description Example
ls List files and directories in the current directory. ls -l
cd Change directory. cd /home/user
pwd Display the current directory. pwd
mkdir Create a new directory. mkdir myfolder
rmdir Remove an empty directory. rmdir myfolder
rm Remove a file or directory. rm -r myfolder
cp Copy files or directories. cp file.txt /backup/
mv Move or rename files. mv oldname.txt newname.txt
cat View the contents of a file. cat file.txt
nano / vi Edit a file in the terminal. nano file.txt
touch Create a new empty file. touch newfile.txt
whoami Show the current logged-in user. whoami
id Display user ID information. id user
adduser Add a new user. adduser john
passwd Change a user's password. passwd john
dnf update Update all system packages. dnf update
dnf remove Remove a package. dnf remove nano
df -h Display disk usage in human-readable format. df -h
free -h Display memory usage. free -h
ping Test network connectivity to a host. ping google.com
ifconfig / ip addr Show network interface information. ifconfig
top Display running processes in real-time. top
htop Interactive process viewer (needs installation). htop
tail View the last lines of a file. tail -f /var/log/messages
chmod Change file permissions. chmod 755 script.sh
chown Change file ownership. chown user:group file.txt
curl Transfer data from or to a server. curl http://example.com
